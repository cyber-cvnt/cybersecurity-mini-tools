# Cybersecurity Mini Tools 🔐

## Overview
This repository contains a collection of beginner-friendly cybersecurity tools built with Python.

These tools demonstrate practical security concepts such as password strength analysis, port scanning, hashing, and log analysis.

---

## Tools Included

### 🔑 Password Strength Checker
Analyzes password complexity and provides feedback.

### 🌐 Port Scanner
Scans open ports on a target system.

### 🔐 Hash Generator
Generates hashes using algorithms like MD5 and SHA256.

### 📊 Log Analyzer
Analyzes log files for suspicious activity.

---

## Technologies Used

- Python
- Networking (Sockets)
- Hashing Algorithms
- File Processing
